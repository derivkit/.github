<h1 align="center">DerivKit</h1>

<p align="center">
  <strong>Efficient, reliable, and well-tested derivatives for scientific forecasting.</strong>
</p>

<p align="center">
  DerivKit provides a unified framework for numerical derivatives, Fisher/DALI expansions,
  and model forecasting — designed for cosmology, but general-purpose across scientific domains.
</p>

## About

DerivKit grew out of practical needs in cosmological inference, combining flexible derivative estimators with rigorous error control and clean, modern APIs.
